# 🎵 快速开始 - 音乐播放器

## 立即开始使用

### 第1步：打开页面
1. 在浏览器中打开 `songs.html`
2. 你会看到105首歌曲的列表

### 第2步：播放歌曲
1. 找到任何你喜欢的歌曲
2. 点击歌曲右侧的 **♫** 按钮（播放图标）
3. 播放器将在页面底部出现

### 第3步：控制播放
- **▶** = 播放/暂停
- **⏮** = 上一首
- **⏭** = 下一首
- **进度条** = 拖动调整进度
- **🔊** = 音量/静音
- **✕** = 关闭播放器

## 📺 播放器界面

```
[♫ 歌曲名 - 歌手名] [⏮] [▶] [⏭] [00:30▓▓▓▓░░░░░░ 02:00] [🔊 ──●─] [✕]
```

## ⚠️ 重要提示

**当前状态**：演示模式
- 所有歌曲都会播放相同的演示音乐文件
- 这是为了演示播放器功能有效

**为了使用真实音乐**，需要：
1. 集成音乐API（网易云、QQ音乐等）
2. 或使用本地音乐文件
3. 参考 `PLAYER_README.md` 获取详细配置说明

## 🔗 音乐源配置方案

### 方案A: 快速 - 使用本地音乐
```
1. 创建 music/ 文件夹
2. 放入 .mp3 文件（命名为 1.mp3, 2.mp3 等）
3. 修改 player.js 中的 getMusicUrl 方法
```

### 方案B: 专业 - 集成网易云音乐 API
```
1. 需要后端服务器
2. 实现歌曲搜索和URL获取
3. 参考 PLAYER_README.md 中的详细步骤
```

### 方案C: 免费 - 使用其他免费音乐源
```
1. 网易云歌单公开分享链接
2. YouTube 音乐
3. Free Music Archive
```

## 🐞 故障排查

### 无声音？
- 检查系统音量
- 检查浏览器音量
- 打开浏览器开发者工具 (F12) 检查错误
- 检查网络连接

### 播放器不显示？
- 刷新页面
- 清除浏览器缓存
- 检查控制台错误信息

### 某些歌曲无法播放？
- 这是正常的（使用演示音乐源）
- 集成真实音乐源后将解决此问题
